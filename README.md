# Sistema de benchmarking com IA
Esse sistema tem como objetivo automatizar e auditar o processo de benchmarking de compras, permitindo que compradores pesquisem fornecedores e produtos no mercado enquanto uma Inteligência Artificial executa uma análise paralela oculta, utilizando dados de mercado e relatórios públicos para sugerir a melhor opção de compra.

Este é o frontend do sistema, ainda em desenvolvimento.
